#### Project Infomation

###### Install Node JS:  
* `sudo apt-get install npm`

###### Install Bower:
* `sudo npm install bower -g`  

###### Run project:
* `cd acuity/src/main/resources/static`
* `bower install angular-ui-router`
* `bower install bootstrap`
* `mvn spring-boot:run`
